\doxysubsection{Description détaillée}
\begin{DoxyVerb}Internal class to parse XML into :class:`~pandas.DataFrame` with third-party
full-featured XML library, ``lxml``, that supports
``XPath`` 1.0 and XSLT 1.0.
\end{DoxyVerb}
