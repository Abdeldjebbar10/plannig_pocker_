\begin{DoxyVerb}Exceptionn raised when performing an operation on non-numerical data.

For example, calling ``ohlc`` on a non-numerical column or a function
on a rolling window.

Examples
--------
>>> ser = pd.Series(['a', 'b', 'c'])
>>> ser.rolling(2).sum()

DataError: No numeric types to aggregate
