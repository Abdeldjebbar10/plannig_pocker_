Description détaillée

Translate slice objects to concatenation along the first axis.

This is the masked array version of `r_`.

See Also
--------
r_

Examples
--------
>>> import numpy as np
>>> np.ma.mr_[np.ma.array([1,2,3]), 0, 0, np.ma.array([4,5,6])]
masked_array(data=[1, 2, 3, ..., 4, 5, 6],
             mask=False,
       fill_value=999999)
