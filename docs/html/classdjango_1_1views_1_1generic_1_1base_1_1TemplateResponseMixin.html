django.views.generic.base.TemplateResponseMixin.get_template_names (   self) 
Return a list of template names to be used for the request. Must return
a list. May not be called if render_to_response() is overridden.

Réimplémentée dans django.views.generic.detail.SingleObjectTemplateResponseMixin, et django.views.generic.list.MultipleObjectTemplateResponseMixin.
