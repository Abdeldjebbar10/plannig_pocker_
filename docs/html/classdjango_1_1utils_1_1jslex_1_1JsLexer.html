Description détaillée

A JavaScript lexer

>>> lexer = JsLexer()
>>> list(lexer.lex("a = 1"))
[('id', 'a'), ('ws', ' '), ('punct', '='), ('ws', ' '), ('dnum', '1')]

This doesn't properly handle non-ASCII characters in the JavaScript source.
